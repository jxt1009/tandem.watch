## Troubleshooting

**Q: I don't see the extension icon in Chrome**
- Make sure you completed the installation steps above
- Try restarting Chrome
- Go to `chrome://extensions/` and look for "tandem.watch" in the list

**Q: The extension says "Not connected"**
- Make sure the person running the server is online
- Check that your room ID is correct
- Try refreshing the Netflix page

**Q: Video/audio isn't working**
- Click the extension icon and allow camera/microphone permissions
- Check that your camera and microphone aren't already in use
- Refresh the page

**Q: Still having issues?**
- Check the Chrome console (press F12, click "Console" tab)
- Ask the technical person who set this up for help

---

## Updating the Extension

When a new version comes out, simply:
1. Go to `chrome://extensions/`
2. Find "tandem.watch"
3. Click the trash/delete icon
4. Reinstall using Option 1 or 2 above
